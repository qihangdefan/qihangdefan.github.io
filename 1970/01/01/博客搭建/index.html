1.下载安装nodejs
https://nodejs.org/zh-cn/

2.安装git
https://git-scm.com/

3.安装hero
修改阿里的npm源: npm config set registry https://registry.npm.taobao.org
mkdir myblog_web
cd myblog_web
npm install hexo

4.初始化hexo项目 —-在新建的文件夹下
hexo init

下载一个自己喜欢的主题，可以从hexo官网下载: https://hexo.io/themes/
我的主题：
github：https://github.com/niemingzhao/niemingzhao.github.io/tree/theme

5.生成静态网站

hexo g
git clone -b theme https://github.com/niemingzhao/niemingzhao.github.io.git themes/default

发布到github
https://github.com/qihangdefan

vim .config.yml

deploy:
type: ‘git’
repo: https://github.com/qihangdefan/qihangdefan.github.io.git

branch: master

输入命令npm install hexo-deployer-git –save 安装自动部署发布工具
输入命令hexo clean && hexo g && hexo d 发布博客，首次发布需要在shell中输入账号和密码。
